Perguntas Frequentes

Quais os critérios para um aluno entrar no nosso banco de dados?

O aluno deve ter renda familiar per capita igual ou menor a meio salário mínimo.

Como eu adoto um aluno?

Você deve preencher o formulário para quem quer adotar alunos.

O que é oferecido ao aluno?

Nós ajudamos os alunos com eletrônicos e/ou internet que ele precise para poder acompanhar as aulas remotamente.

Não tenho dinheiro, posso ajudar de alguma forma?

Nós aceitamos voluntários que possam arrumar os eletrônicos e/ou doar eletrônicos.

Após a pandemia, esse programa continuará?

Se a maior parte dos adotantes quiserem continuar nessa campanhã nos pretendemos continuar com este projeto.
